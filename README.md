# Tic Tac Toe — Python OOP Project

A fully functional Tic Tac Toe game built in Python using 
object oriented programming with a graphical interface, 
background music and sound effects.

## About
Built during my second semester as part of my Object Oriented 
Programming course. This project was a step forward from my 
first semester work — moving from console based C++ applications 
to building a complete GUI based game in Python with real time 
audio and interactive gameplay.

## Features
- Single player mode with computer opponent
- Multiplayer mode for two players
- Background music during gameplay
- Sound effects for clicks, wins and draws
- Restart functionality with full game reset
- Clean and interactive graphical interface built with Tkinter

## OOP Concepts Used
- Class and object creation
- Encapsulation of game logic and UI in a single class
- Constructor initialization with __init__
- Method definitions for game flow, win checking and UI updates
- Event driven programming with button bindings

## Tech Stack
- Language: Python
- GUI Library: Tkinter
- Audio Library: Pygame
- IDE: VS Code

## Requirements
- Python 3.11
- Pygame library

## Installation
1. Install Python 3.11 from python.org
2. Install pygame by running:
   pip install pygame
3. Clone the repository
4. Run the game:
   python OEL_PROJECT.py
